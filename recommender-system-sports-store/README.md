## Реализация

Рекомендательная система реализована на Python и работает в консольном режиме.
В зависимости от количества оценок пользователя применяется один из подходов:

- **Correlation-based (k-NN + разреженная матрица)** — если у пользователя есть несколько оценок
- **Content-based (TF-IDF + cosine similarity)** — если данных о пользователе недостаточно

### Correlation-based рекомендации
- Формируется разреженная матрица предпочтений (CSR)
- Используется алгоритм k-ближайших соседей (NearestNeighbors)
- Метрика сходства — косинусная

### Content-based рекомендации
- Текстовые описания товаров векторизуются с помощью TF-IDF
- Схожесть товаров вычисляется через cosine similarity
- Рекомендации формируются на основе схожести описаний
